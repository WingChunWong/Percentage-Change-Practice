# 百分比變化練習 (Percentage Change Practice)

這是一個互動式的百分比變化練習工具

## 📋 專案介紹

本工具透過視覺化的界面呈現百分比變化問題，包含三種常見題型：
- 已知原值與百分比變化，計算新值
- 已知新值與百分比變化，計算原值
- 已知原值與新值，計算百分比變化

使用者可即時提交答案並獲得反饋，系統會自動記錄分數與練習進度，幫助追蹤學習成效。

## ✨ 功能特點

1. **多語言支援**：可切換中英文顯示，適應不同語言需求
3. **即時反饋**：提交答案後立即顯示正確與否，幫助即時糾錯
4. **進度追蹤**：透過分數與進度條可直觀了解學習進度，目標分數為150分
5. **本地存儲**：自動保存練習狀態，重新打開頁面可繼續練習
6. **成績報告**：可查看總題數、正確率等統計數據，方便回顧學習成果

## 📱 界面預覽

- 主界面：顯示題目、輸入區與狀態欄
- 多語切換：右上角按鈕切換中英文顯示
- 成績報告：展示分數、正確率等統計數據
